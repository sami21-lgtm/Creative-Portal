# 📖 Studies | Student & Admin Dashboard

Eti ekti modern, responsive ebong interactive dashboard project. Eti muloto university student-der academic progress track korar jonno design kora hoyeche.

## 🚀 Features

- **Dynamic Analytics:** Chart.js bebohar kore Pass Rate, Difficulty Levels, ebong Weekly Performance graph akare dekha jay.
- **Multi-language Support:** Dashboard-ti English, Bangla, ebong Arabic—ei tinti bhashay switch kora jay.
- **Interactive Calendar:** Student-der schedule maintain korar jonno ekti grid-based calendar.
- **Navigation System:** Sidebar bebohar kore Dashboard, Calendar, Courses, Leaderboard, ebong Notifications-e switch kora jay.
- **GitHub Ready:** Eti single-file ba separate file (HTML/CSS/JS) hishebe GitHub Pages-e host korar jonno optimized.

## 🛠️ Technologies Used

- **HTML5:** Project structure-er jonno.
- **CSS3:** Modern UI/UX ebong responsive layout-er jonno.
- **JavaScript (ES6):** Language switching, navigation, ebong data management-er jonno.
- **Chart.js:** Interactive charts ebong graphs-er jonno.
